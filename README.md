# Stock Trading Simulator

This project was created with Django, Yahoo Finance API, Plotly graphs, SQL, and AJAX. Hosted on Heroku.

### To View

Open [StockMatrix](https://stockmatrix.herokuapp.com/) to view it in the browser.
